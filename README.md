# 🚀 TradeX Frontend

A modern, responsive **admin dashboard UI** for the TradeX trading and quantitative analysis platform.
Built with **React, Vite, TailwindCSS, and advanced UI libraries**, this frontend provides powerful visualization and control over trading strategies, models, backtesting, market data, and sentiment analysis.

---

## 🧩 Features

### 📊 Dashboard

* Overview of key metrics (Strategies, Models, Backtests, Data Sources)
* Performance charts (PnL trends)
* Quick action buttons

---

### 🧠 Strategies

* View strategies with:

  * Name
  * Symbol
  * Time Horizon
  * Indicators & Patterns
* Filter by:

  * Symbol
  * Time Horizon
* Expandable detailed view
* Smooth animations and transitions

---

### 🤖 Models

* Model performance metrics:

  * Long/Short Trades
  * Win/Loss Trades
  * Win Rate / Loss Rate
  * Max Drawdown
  * Consecutive Wins/Losses
* Interactive charts on model selection
* Visual progress indicators

---

### 🧪 Backtesting

* Select:

  * Strategy
  * Exchange
* Run backtests dynamically
* Results include:

  * Ledger details
  * Win vs Loss chart
  * PnL per trade visualization

---

### 📡 Data

* Select Exchange and Coin
* Fetch market data
* Interactive **candlestick chart** (OHLCV)
* Real-time style visualization

---

### 💬 Sentiment Analysis

* Reddit sentiment analysis interface
* Analyze posts & comments
* Displays:

  * Sentiment score
  * Label (Positive / Neutral / Negative)
  * Confidence
* Hourly sentiment analytics
* Post/comment filtering
* Live feed UI

---

## 🎨 UI/UX Highlights

* 🌗 Dark & Light Theme support
* ⚡ Smooth animations (via Motion)
* 📱 Fully responsive design
* 📊 Interactive charts (Recharts)
* 🧱 Modular component architecture
* 🎯 Clean and modern fintech UI

---

## 🛠️ Tech Stack

* **Frontend Framework:** React 18 + Vite
* **Styling:** Tailwind CSS v4
* **UI Libraries:**

  * Material UI (MUI)
  * Radix UI
* **Charts:** Recharts
* **Animations:** Motion
* **Theme Management:** next-themes
* **Form Handling:** React Hook Form

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/MY-Raza/TradeX_Frontend.git
cd TradeX_Frontend
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run development server

```bash
npm run dev
```

App will be available at:

```
http://localhost:5173/
```

---

## 🔌 Backend Integration

This frontend is designed to connect with the **TradeX backend** for:

* Strategy execution
* Model predictions
* Backtesting engine
* Market data APIs
* Sentiment analysis services

---

## ⚠️ Important Notes

* Ensure Node.js is installed and available in PATH
* Do not run frontend inside Python virtual environments
* Use separate terminals for frontend and backend
